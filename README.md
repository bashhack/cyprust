## CLI Utilities

### paiku
A customizable haiku generator.

"country name or coordinates (region), happy (mood), nature (environment)"

- region-detection/geolocation
- sentiment analysis
- i18n support


### directors-cut
A generative cinematic remixer, empowering you to create the movie no one knew they needed.

"[Star Wars/LOTR/The Matrix/Toy Story] (trilogy) directed by [Martin Scorcese, Sam Mendes, Stanley Kubrick, Alejandro G. Inarritu] (director) on [ayahuasca, cocaine, lsd] (drug)"

- http://stanza.readthedocs.io/en/latest/index.html
- 'create your own scene' via Markov Chain
- syntax substitution


### kepler
A planetary orbit simulator.

- by datetime
- find closest distance between any two celestial bodies


### parenthetical
A homebrew lisp evaluator.
